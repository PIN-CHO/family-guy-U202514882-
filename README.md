Family Guy 主题网页项目 README
一、项目概述
本项目是基于美国现象级动画《恶搞之家》（Family Guy）开发的粉丝向角色展示网页，旨在为动画爱好者提供直观的角色信息查询、情感共鸣与互动交流平台。网页深度融入“自由、随性且充满趣味”的美式文化内核，以淡蓝绿色为主色调，搭配水彩风背景插画，打造清新明亮的视觉体验，同时通过简洁的导航逻辑与丰富的交互效果，提升用户浏览体验。

二、技术栈与开发环境

1. 核心技术

- 前端基础：HTML5（语义化标签构建页面结构）、CSS3（Flex布局、响应式设计、动画效果）、JavaScript（交互逻辑实现、音频控制）

- 辅助工具：AI工具（Copilot 代码优化、豆包生成水彩风背景插画与角色素材）

- 版本控制：Git（项目代码管理与备份）

- 文档工具：LaTeX（实验报告撰写，提升文档专业性）

2. 开发与运行环境

- 操作系统：Windows 10/11、macOS 10.15+

- 浏览器：Chrome 100+、Edge 100+、Firefox 95+（支持CSS3与ES6语法）

- 开发工具：VS Code（推荐安装HTML CSS Support、JavaScript and TypeScript Nightly插件）

- 运行方式：本地直接打开主页面HTML文件，无需搭建服务器

三、项目架构设计

1. 整体架构：“1+9”树状二级结构

项目采用核心聚合、分支辐射的树状架构，以主页面为唯一入口，9个角色页为平行二级节点，页面间通过链接实现互联，形成完整浏览闭环
